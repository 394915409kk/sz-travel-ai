# 09 版本记录

## 记录规则

- 能力中心结构、规则、SOP、安装记录和安全边界发生变化时，应更新本文件。
- 版本号建议采用 `主版本.次版本.修订号`。
- 未正式启用的内容标记为 `草案`。
- 每条记录应说明变更范围、验证状态和兼容性影响。

## 版本模板

```text
版本：

状态：

验证结果：
回滚说明：
```
