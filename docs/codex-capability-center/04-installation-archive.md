# 04 安装归档

## 定位

安装归档用于记录 Skills、插件、连接器、运行依赖和辅助工具的来源、版本、验证和回滚信息。

## 归档字段

- 安装对象
- 类型
- 来源
- 安装日期
- 安装位置
- 版本或 commit
- 执行命令
- 验证结果
- 回滚方式
- 当前状态
- 备注

## 安装流程

1. 确认来源可信和适用范围。
2. 记录安装前状态。
3. 优先在隔离环境或 dry-run 模式验证。
4. 执行安装。
5. 运行结构、依赖和功能检查。
6. 记录结果和回滚方式。
7. 人工确认后进入可用状态。

## 安全要求

- 不在归档中记录真实密码、token 或生产连接。
- 不执行来源不明的安装脚本。
- 不自动同步到业务仓库。
- 安装失败时停止扩散，并进入故障复盘。
