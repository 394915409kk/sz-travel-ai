# 01 Skills 中心

## 定位

Skills 中心用于登记可复用能力、触发条件、输入输出、验证方式和安全限制。

## Skill 登记字段

- Skill 名称
- 适用场景
- 触发关键词
- 所需输入
- 标准输出
- 依赖工具
- 所需权限
- 禁止事项
- 验证命令
- 维护人
- 当前版本

## 生命周期

1. 提出能力需求。
2. 编写或引入 Skill。
3. 在隔离环境完成验证。
4. 记录安装来源和版本。
5. 人工确认后启用。
6. 定期复测、升级或停用。

## 验收标准

- 触发范围清晰，不与其他 Skill 冲突。
- 输出结构稳定，可被人工复核。
- 不包含真实密钥、客户隐私或生产连接。
- 高风险操作默认只读或要求显式确认。
- 有失败处理、回滚或停止说明。
