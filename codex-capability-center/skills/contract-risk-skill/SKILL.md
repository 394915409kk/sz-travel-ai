# Contract Risk Skill

## 触发场景

- 审阅供应商、客户、地接、酒店、机票、OTA、B2B、企业团合同。
- 梳理付款、退款、取消、不可抗力、违约、责任限制、用印和发票风险。
- 输出合同风险意见、修改建议或谈判清单。

## 执行流程

1. 确认合同类型、签约主体、服务范围、金额、付款节点和履约周期。
2. 逐条检查核心风险：价格、资源、取消、退款、赔付、保险、不可抗力、争议解决。
3. 标记高风险条款、缺失条款、模糊条款和需法务确认项。
4. 给出业务可理解的修改建议和谈判话术。
5. 明确这不是最终法律意见。

## 禁止事项

- 不代替律师出具最终法律意见。
- 不替用户确认盖章、付款、合同定稿或对外承诺。
- 不忽略签约主体、授权代表、附件、补充协议和版本日期。
- 不用营销语言掩盖履约责任。

## 完成标准

- 风险分级清楚：高、中、低。
- 每个风险有条款位置、影响和建议改法。
- 需法务、财务或业务负责人确认的事项明确。

```markdown
- 高风险：

```
