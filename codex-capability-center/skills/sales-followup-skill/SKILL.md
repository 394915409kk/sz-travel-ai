# Sales Followup Skill

## 触发场景

- 客户咨询后需要分级、跟进话术、二次触达或促单方案。
- 销售需要报价回复、异议处理、催定金、复购或转介绍脚本。
- 分析线索流失原因、转化率和销售动作质量。

## 执行流程

1. 识别客户类型、需求强度、预算、人数、日期、决策人和时间窗口。
2. 将线索分为 A/B/C/D，并给出下一步动作。
3. 话术先解决信任和匹配，再报价或推动确认。
4. 对价格、库存、政策、合同、退款等不确定内容使用待确认表达。
5. 输出销售可复制话术和跟进节奏。

## 禁止事项

- 不承诺未核实价格、房态、航班、签证、退款或特殊服务。
- 不诱导客户提供不必要隐私。
- 不使用恐吓、虚假稀缺或虚假成交截图。
- 不绕开合同、收款、发票和公司审批流程。

## 完成标准

- 客户分级清楚，下一步动作具体。
- 话术自然、可发送、风险边界清楚。
- 待核价、待库存、待政策确认事项已标记。
- 能被销售主管用于复盘转化动作。

## 输出格式

```markdown
## 销售跟进
- 客户等级：
- 当前判断：
- 本次回复：
- 下一次触达：
- 待确认：
```
